# Description du GitHub
Nous sommes trois étudiants en Master 1 SID
L’objectif de ce projet est de développer un système d’aide à la décision élaboré par des analyses détaillées sur des points d'intérêts à partir de la visualisation de données textuelles issues d’une base de données (BDD) en ligne. 

Notre problématique est la suivante : Les catastrophes naturelles sont-elles plus fréquentes et destructrices depuis le début du XXIe siècle? 

Vous trouverez dans le GitHub les différents codes de collectes de données, nettoyage ainsi que les scripts pour la Base de Données
